# Faultspan Demo Runbook

## Before the session

1. Start the API and confirm `/ready`.
2. Start the web application and confirm `/api/health`.
3. Confirm the configured contract address using a view call.
4. Fund the demo wallets with Studionet faucet GEN.
5. Open `design/wireframes.html` only if judges ask about design exploration.
6. Keep a backup recording available; label it as recorded.

## Three-minute proof

1. Open case `FS-2026-0042` and state the root obligation.
2. Show the graph: research, analysis, and writing are separate commitments.
3. Select Analysis and show the obligation, evidence, x402 record, and bond rule together.
4. Explain that GenLayer decides the finding, while deterministic code calculates the money.
5. Open Settlement and show conservation: `11 = 8 returned + 3 recovered`.
6. Switch to Ledger to demonstrate the non-graph representation.
7. Open Add evidence and show wallet-signed content addressing without submitting sensitive data.

## Live on-chain extension

If the deployment and funded accounts are verified, create a fresh case from the wizard and show submitted → accepted → finalized. If any prerequisite is missing, save the local draft and say explicitly that no transaction was submitted.
